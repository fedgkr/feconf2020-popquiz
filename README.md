## 점수 측정 방식

- 세션 문제 당 1점 (총 12점 만점)
- 보너스 문제 당 1점 (단, 보너스 문제는 만점 미만일 경우에만 적용)

> ex) 한 트랙에 대한 문제를 모두 맞히고 (6점) 보너스 문제를 전부 맞췄을 경우 (7점) => 13점이 되므로 **12점**으로 인정
